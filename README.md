# BTCStake — Bitcoin tBTC Staking on OP_NET

The first native Bitcoin staking protocol built on OP_NET Bitcoin L1.

## 🔗 Live Demo
https://btc-staking-alpha.vercel.app

## 📋 Features
- ✅ Stake tBTC and earn 12% APY
- ✅ Unstake anytime with real on-chain transaction
- ✅ Claim rewards directly to wallet
- ✅ Live TVL, BTC price, active stakers
- ✅ Reward calculator
- ✅ Personal dashboard
- ✅ Analytics page
- ✅ Referral system
- ✅ Synthwave 3D UI
